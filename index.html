``` --- ## README.md ### Private Journal A **single-file, offline-first private journal** designed for speed, clarity, and long-term personal use. ### Core Principles * Privacy-first (localStorage only, no sync, no server) * Minimal UI, distraction-free * Fast capture * Plain text longevity ### Features * One `index.html` * Automatic local-time timestamps * Inline **#tags** support (plain text) * Mark notes as important (★ or long-press on mobile) * Search across notes and tags * Copy, delete, expand entries * Import / Export backup * Installable as PWA (local use) ### Tags (Important) Tags are **inline text**, not UI elements. **Best practice:** place tags in the **first line** so they are always visible in the preview. Example: ``` #work #finance Reviewed quarterly budget ``` Tags are searchable automatically. ### How to Use * Type note → press **Enter** to save * `Shift + Enter` for new line * Use `#tags` anywhere in text * Long‑press a note on mobile to mark important ### Data * Stored locally in your browser * Export regularly for backup --- **FINAL STABLE – UI locked**
