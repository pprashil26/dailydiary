# Private Journal v1.2.2

A simple, offline, privacy-focused personal journal app. All data is stored locally in your browser. No account, no cloud, no syncing. Designed for speed, minimalism, and usability.

---

## Core Principles
- **Privacy First:** All notes are stored locally in your browser. Nothing is sent to any server.  
- **Offline Friendly:** Works completely offline. Install as a PWA for app-like experience.  
- **Minimal & Efficient:** Quick note-taking with compact UI. Focus on writing, not distractions.

---

## Features
- Create new notes with **automatic timestamp** (local device time).  
- **Bold** and **CAPS** toggle for new notes.  
- Assign **tags** to notes (inline).  
- Mark notes as **important** with ⭐ (long-press on mobile).  
- **Focus mode:** hide all existing notes to focus on writing a new note.  
- **Search:** filter notes quickly.  
- **Copy** and **delete** individual notes.  
- Expand long notes for full view.  
- Export and import your journal as a `.txt` file.  
- Fully responsive for mobile devices while preserving desktop UI.  

---

## How to Use
1. Open `index.html` in your browser.  
2. Type your note in the **new note** box.  
   - **Shift+Enter** → line break  
   - **Enter** → save note  
3. Use the buttons next to the new note box:  
   - **+** → add note  
   - **B** → bold  
   - **A** → CAPS  
   - **⏰** → insert current timestamp  
4. Mark notes important with ⭐ or toggle focus mode.  
5. Search notes using the search box at the top.  
6. Copy, delete, or expand notes using the respective icons.  
7. Backup or restore your journal using **Export** / **Import**.  

---

## Installation as PWA
- Open in Chrome or Edge → click menu → **Install** → works like an offline app.
